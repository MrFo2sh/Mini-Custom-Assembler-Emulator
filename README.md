# Mini-Custom-Assembler-Emulator
Custom Assebly Language execution simulator
Syntax Example:

section.data
anylabel dd 10
1test dd 5
fouad dd 7
ahmed dd 4
section.text
mov eax , 10D
add edx , eax
mul edx , 2D
mov ebx , 4D
div ebx , 2D
add eax , ahmed
mul ebx , fouad
div eax , ahmed
mov ebx , 1test
add edx , anylabel
