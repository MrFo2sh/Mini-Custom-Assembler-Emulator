# Mini-Custom-Assembler-Emulator
Custom Assebly Language execution simulator
<br/>
Put those commands in a file.txt
<br/>
Syntax Example:
<br/>
section.data
<br/>
anylabel dd 10
<br/>
1test dd 5
<br/>
fouad dd 7
<br/>
ahmed dd 4
<br/>
section.text
<br/>
mov eax , 10D
<br/>
add edx , eax
<br/>
mul edx , 2D
<br/>
mov ebx , 4D
<br/>
div ebx , 2D
<br/>
add eax , ahmed
<br/>
mul ebx , fouad
<br/>
div eax , ahmed
<br/>
mov ebx , 1test
<br/>
add edx , anylabel
